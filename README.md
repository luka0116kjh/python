#  파이썬 인증 (Python Auth)

간단한 CLI 기반 사용자 인증 시스템입니다.

##  기능
- **회원가입**: 안전한 비밀번호 해싱 (PBKDF2 + Salt).
- **로그인**: 보안 인증.
- **저장소**: JSON 파일 (`users.json`).

##  사용법
```bash
python main.py
```
1. **회원가입**: 새 계정 생성.
2. **로그인**: 아이디/비밀번호로 접속.
3. **종료**: 프로그램 종료.

##  파일
- `main.py`: 메인 스크립트.
- `users.json`: 사용자 데이터베이스 (자동 생성됨).
